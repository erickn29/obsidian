Посмотреть только первую строку файла `file.txt`:

```
head -n1 file.txt
```

Посмотреть первые 2 строки файла `file.txt`:

```
head -n2 file.txt
```

Посмотреть одну последнюю строку файла `file.txt`:

```
tail -n1 file.txt
```

Вывести по каждой строке файла `file.txt` только первые 5 символов:

```
cut -c 1-5 file.txt
```
